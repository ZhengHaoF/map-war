# 开发规范

## 项目概述

本项目是一个中国战争 AI 游戏，使用 Vue 3 + Pixi.js 实现地图交互。

### GameButton

深色半透明按钮，支持激活和危险状态。

```vue
<GameButton :active="isActive" :danger="isDanger" @click="handler">
  按钮文本
</GameButton>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| active | Boolean | false | 激活状态（蓝色高亮） |
| danger | Boolean | false | 危险状态（红色悬停） |

---

### GameContextMenu

右键菜单，支持多项目和危险变体。

```vue
<GameContextMenu
  :visible="showMenu"
  :position="{ x: 100, y: 200 }"
  :items="[
    { action: 'info', label: '查看信息' },
    { action: 'delete', label: '删除', danger: true }
  ]"
  @select="onSelect"
/>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| visible | Boolean | false | 是否显示 |
| position | Object | `{ x: 0, y: 0 }` | 菜单位置 |
| items | Array | `[]` | 菜单项 `{ action, label, danger? }` |

| Event | 参数 | 说明 |
|-------|------|------|
| select | action: string | 点击菜单项触发 |

---

### GameModal

弹窗/浮动面板组件，自动 Teleport 到 body。支持普通居中弹窗和可拖动的浮动面板两种模式。

```vue
<!-- 普通弹窗（居中、带遮罩） -->
<GameModal :visible="showModal" title="标题" @close="showModal = false">
  <p>内容</p>
</GameModal>

<!-- 可拖动的浮动面板（无遮罩，自由定位） -->
<GameModal
  :visible="showPanel"
  title="面板"
  :draggable="true"
  :overlay="false"
  :init-x="160"
  :init-y="160"
  @close="showPanel = false"
>
  <p>可拖动的面板内容</p>
</GameModal>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| visible | Boolean | false | 是否显示 |
| title | String | `''` | 标题文本 |
| minWidth | String | `'280px'` | 最小宽度 |
| draggable | Boolean | false | 是否可拖动 |
| overlay | Boolean | true | 是否显示遮罩层 |
| initX | Number | 160 | 初始 X 位置（`draggable` 时生效） |
| initY | Number | 160 | 初始 Y 位置 |

| Event | 说明 |
|-------|------|
| close | 关闭弹窗 |

**拖动说明：** 按住标题栏拖拽，标题栏在 `draggable` 时显示 `grab/grabbing` 光标。`overlay=false` 时遮罩透明且不阻塞地图交互。

## 开发约定

1. **组件命名**：通用组件以 `Game` 前缀命名
2. **样式隔离**：使用 `scoped` 避免样式污染
3. **颜色格式**：PixiJS 使用 `0xRRGGBB`，CSS 使用 `#RRGGBB`
4. **事件命名**：使用 kebab-case（如 `@select`、`@close`）
5. **Prop 类型**：使用对象形式定义，包含 type、default、required

## 动画函数库

文件位置：`src/utils/troopAnimation.js`

### playArcAnimation - 弧线动画（通用）

从起点到终点的弧线动画，支持两种模式和文字弹出。

**模式：**
- `mode: 'dots'` - 多箭头沿弧线移动（适合派兵、行军）
- `mode: 'orb'` - 光球沿弧线移动（适合宣战、施法）

**效果组合：**
- `explosion: true` - 到达后爆炸+冲击波
- `text: '文字'` - 到达后在目的地上方弹出飘字

```js
import { playArcAnimation } from '@/utils/troopAnimation'

// 派兵模式（箭头 + 飘字）
await playArcAnimation({
  from: { x: 500, y: 300 },
  to: { x: 600, y: 400 },
  container: worldContainer,
  mode: 'dots',
  text: '出兵！',
  color: 0xffcc00,
  dots: 5,
  duration: 2000,
})

// 宣战模式（光球 + 爆炸 + 飘字）
await playArcAnimation({
  from: { x: 500, y: 300 },
  to: { x: 600, y: 400 },
  container: worldContainer,
  mode: 'orb',
  explosion: true,
  shockwaves: 3,
  text: '宣战！',
  color: 0xff4444,
  duration: 1200,
  explosionDuration: 800,
})
```

**参数说明：**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| from | `{x, y}` | - | 起点屏幕坐标 |
| to | `{x, y}` | - | 终点屏幕坐标 |
| container | Container | - | PixiJS 容器 |
| mode | `'dots'\|'orb'` | `'dots'` | 动画模式 |
| explosion | boolean | `false` | 是否爆炸 |
| shockwaves | number | `3` | 冲击波层数 |
| text | string | - | 弹出文字（可选） |
| textColor | number | `0xffffff` | 弹出文字颜色 |
| highlightGfx | Graphics | - | 高亮图形对象 |
| fromFeature | Feature | - | 出发地Feature |
| toFeature | Feature | - | 目的地Feature |
| onHighlight | function | - | 高亮回调 |
| color | number | `0xffcc00` | 动画颜色 |
| dots | number | `5` | 箭头数量（dots模式） |
| spacing | number | `0.08` | 箭头间距（dots模式） |
| duration | number | `2000` | 飞行时长（毫秒） |
| explosionDuration | number | `800` | 爆炸时长（毫秒） |

### playScoutAnimation - 探察动画

从起点向外扩散圆环，表示侦察。

```js
import { playScoutAnimation } from '@/utils/troopAnimation'

await playScoutAnimation({
  from: { x: 500, y: 300 },
  container: worldContainer,
  color: 0x22c55e,
  rings: 3,
  duration: 1500,
  text: '侦察！',
})
```

### playBattleAnimation - 战斗动画

两城之间交火，双向发射光点。

```js
import { playBattleAnimation } from '@/utils/troopAnimation'

await playBattleAnimation({
  from: { x: 500, y: 300 },
  to: { x: 600, y: 400 },
  container: worldContainer,
  highlightGfx: gfx,
  fromFeature: featureA,
  toFeature: featureB,
  onHighlight: highlightFn,
  colorA: 0x3b82f6,
  colorB: 0xef4444,
  duration: 2500,
  text: '交战！',
})
```

### 弹出文字说明

所有动画函数都支持 `text` 参数，在动画结束时在目的地上方弹出飘字：
- 从目标位置上方 30px 开始
- 向上飘动 40px
- alpha 从 1 渐变到 0
- 持续约 1.2 秒后自动销毁
