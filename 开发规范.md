# 开发规范

## 项目概述

本项目是一个中国战争 AI 游戏，使用 Vue 3 + Pixi.js 实现地图交互。

---

## 目录结构

```
src/
├── components/
│   ├── ui/                    # 通用 UI 组件库
│   │   ├── GameButton.vue     # 深色游戏按钮
│   │   ├── GameContextMenu.vue # 右键菜单
│   │   ├── GameModal.vue      # 弹窗
│   │   ├── InfoTable.vue      # 键值对信息表
│   │   └── LegendPanel.vue    # 图例面板
│   └── LeafletMap.vue         # 地图主组件
├── data/
│   ├── owners.ts              # 政权枚举
│   └── ownerColors.ts         # 政政权颜色映射
├── views/
│   └── MapView.vue            # 地图页面
├── router/
│   └── index.js               # 路由配置
└── stores/
    └── counter.js             # 状态管理
```

| 层级 | 用途 |
|------|------|

### GameButton

深色半透明按钮，支持激活和危险状态。

```vue
<GameButton :active="isActive" :danger="isDanger" @click="handler">
  按钮文本
</GameButton>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| active | Boolean | false | 激活状态（蓝色高亮） |
| danger | Boolean | false | 危险状态（红色悬停） |

---

### GameContextMenu

右键菜单，支持多项目和危险变体。

```vue
<GameContextMenu
  :visible="showMenu"
  :position="{ x: 100, y: 200 }"
  :items="[
    { action: 'info', label: '查看信息' },
    { action: 'delete', label: '删除', danger: true }
  ]"
  @select="onSelect"
/>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| visible | Boolean | false | 是否显示 |
| position | Object | `{ x: 0, y: 0 }` | 菜单位置 |
| items | Array | `[]` | 菜单项 `{ action, label, danger? }` |

| Event | 参数 | 说明 |
|-------|------|------|
| select | action: string | 点击菜单项触发 |

---

### GameModal

弹窗组件，自动 Teleport 到 body。

```vue
<GameModal :visible="showModal" title="标题" @close="showModal = false">
  <p>内容</p>
</GameModal>
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| visible | Boolean | false | 是否显示 |
| title | String | `''` | 标题文本 |
| minWidth | String | `'280px'` | 最小宽度 |

| Event | 说明 |
|-------|------|
| close | 关闭弹窗（点击遮罩或关闭按钮） |

---

### InfoTable

键值对信息展示表格。

```vue
<InfoTable :rows="[
  { label: '政权', value: '国民政府' },
  { label: '工业', value: '7 / 10' }
]" />
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| rows | Array | `[]` | 数据行 `{ label, value }` |

---

### LegendPanel

图例面板，固定在左下角。

```vue
<LegendPanel :items="[
  { label: '国民政府', color: '#3b82f6' },
  { label: '中共苏区', color: '#ef4444' }
]" />
```

| Prop | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| items | Array | `[]` | 图例项 `{ label, color }` |

## 开发约定

1. **组件命名**：通用组件以 `Game` 前缀命名
2. **样式隔离**：使用 `scoped` 避免样式污染
3. **颜色格式**：PixiJS 使用 `0xRRGGBB`，CSS 使用 `#RRGGBB`
4. **事件命名**：使用 kebab-case（如 `@select`、`@close`）
5. **Prop 类型**：使用对象形式定义，包含 type、default、required
