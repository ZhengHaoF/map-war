# 1931 年 — 世界与中国的十字路口

> 本文档供 AI 决策参考，描述了 1931 年的国际格局与中国国内形势。

---

## 一、国际格局

### 1. 全球大萧条

1929 年开始的大萧条在 1931 年达到深度低谷。全球贸易萎缩、失业率飙升、各国转向贸易保护主义和内向型经济。这深刻影响了列强在远东的利益分配和干预意愿。

### 2. 日本 — 军部抬头

- **国内**：大萧条重创日本经济，农村极度贫困，军部（尤其是关东军）势力膨胀，主张通过侵占中国东北转嫁危机。
- **关东军**：驻扎在中国东北的日本军队（约 2 万人），实际上不听东京内阁节制，自行策划军事行动。
- **9 月 18 日**：关东军自行炸毁沈阳柳条湖附近南满铁路，诬陷中国军队，随即进攻沈阳北大营——**九一八事变**爆发。
- **结果**：日军在数月内占领辽宁、吉林、黑龙江三省全境。

### 3. 苏联

- 正在进行第一个五年计划，工业快速崛起。
- 在远东地区部署大量兵力，与关东军形成对峙。
- 对日本侵华持警惕但避免直接冲突的立场。
- 通过共产国际与中共保持联系，但实际援助有限。

### 4. 欧美列强

- **美国**：胡佛政府深陷国内危机，对远东事务采取孤立主义，仅发表"不承认主义"（史汀生主义）。
- **英国**：经济困难，无力干预远东，对日本采取绥靖态度。
- **国联**：派出李顿调查团，但调查报告要到 1932 年才出炉，实际无约束力。

## 二、中国国内局势

### 1. 国民政府（南京）

| 项目 | 内容 |
|------|------|
| 领导人 | 蒋介石（军事委员会委员长） |
| 控制区 | 长江中下游各省、东南沿海 |
| 军队 | 中央军（精锐约 30 万） |
| 特点 | 名义上统一全国，实际仅控制江南核心区 |

- **攘外必先安内**：蒋介石的基本国策，优先剿共，对日暂取忍让。
- **中央军**：德械装备，是当时中国最现代化的军队。
- **财政**：依靠江浙财阀和关税，受大萧条冲击严重。

### 2. 中共苏区

| 项目 | 内容 |
|------|------|
| 领导人 | 毛泽东、朱德、周恩来 |
| 控制区 | 江西瑞金为中心的中央苏区，以及鄂豫皖、湘鄂西等 |
| 军队 | 红军（约 10 万，装备极差） |
| 特点 | 土地革命、农村包围城市 |

- **1931 年 11 月**：在瑞金宣布成立**中华苏维埃共和国**，毛泽东任主席。
- **三次反围剿**：1930 年底至 1931 年 9 月，红军连续粉碎国民政府三次围剿。
- **经济**：苏区极度贫困，依靠分田地动员农民、打土豪筹款。
- **装备**：步枪为主，弹药匮乏，无空军和重炮。

### 3. 东北军

| 项目 | 内容 |
|------|------|
| 领导人 | 张学良 |
| 控制区 | 东北三省（九一八前） |
| 军队 | 约 30 万，含海空军 |
| 特点 | 中国最现代化的地方军队 |

- 九一八事变时，张学良奉行不抵抗政策，日军快速占领沈阳、长春、吉林等城市。
- 至 1931 年底，东北军主力退回关内，驻防平津和河北。
- 东北义勇军开始自发抵抗，但缺乏统一指挥和补给。
- 失去东北工业基地后，东北军士气低落，寄人篱下。

### 4. 晋系

| 项目 | 内容 |
|------|------|
| 领导人 | 阎锡山 |
| 控制区 | 山西、绥远大部 |
| 军队 | 晋军（约 15 万） |
| 特点 | 独立王国、精于经营 |

- 阎锡山在中原大战（1930）失败后下野，1931 年 8 月重返山西，重新掌控军政。
- 山西有相对完整的军工体系（太原兵工厂）和铁路网。
- 对日军和国民政府两面周旋，保存实力为第一要务。
- 阎锡山治下的山西社会相对稳定，有"模范省"之称。

### 5. 桂系

| 项目 | 内容 |
|------|------|
| 领导人 | 李宗仁、白崇禧 |
| 控制区 | 广西全省 |
| 军队 | 桂军（约 10 万） |
| 特点 | 军政合一、民团制度 |

- 桂系在中原大战中反蒋失败后退回广西，励精图治。
- 白崇禧推行"三自三寓"政策（自治、自卫、自给），广西成为"模范省"。
- 桂军以民团为基础，动员能力强，山地作战经验丰富。
- 与国民政府保持表面服从，实际半独立状态。

### 6. 川军

| 项目 | 内容 |
|------|------|
| 领导人 | 刘湘、刘文辉等 |
| 控制区 | 四川全省（军阀割据混战） |
| 军队 | 川军各部（合计约 20 万） |
| 特点 | 内部混战、装备最劣 |

- 四川处于多派军阀混战状态（刘湘 vs 刘文辉的"二刘之战"）。
- 川军装备极差（步枪型号杂乱、缺乏重武器），但兵员充足、山地适应性强。
- 1931 年二刘尚未决出胜负，四川处于分裂状态。
- 由于秦岭阻隔，外部势力难以直接干预四川。

### 7. 马家军

| 项目 | 内容 |
|------|------|
| 领导人 | 马步芳、马鸿逵、马麟 |
| 控制区 | 甘肃、青海、宁夏 |
| 军队 | 回族骑兵为主（约 5 万） |
| 特点 | 宗教结合、骑兵精锐 |

- 马家军是回族军阀，以骑兵闻名，机动性强。
- 控制西北回民聚居区，与中央政府关系若即若离。
- 经济落后，兵源以回民为主，装备一般但战斗意志顽强。
- 对红军和中央军都保持警惕。

### 8. 新疆

| 项目 | 内容 |
|------|------|
| 领导人 | 金树仁 |
| 控制区 | 新疆全省 |
| 军队 | 省军（约 2 万） |
| 特点 | 地广人稀、民族复杂 |

- 金树仁统治腐败，民族矛盾尖锐（汉、维、回、哈萨克冲突）。
- 1931 年爆发**哈密起义**，维吾尔族反抗金树仁统治。
- 苏联在新疆有重大影响，暗中支持分裂势力。
- 新疆距离内地遥远，中央势力几乎无法到达。

### 9. 西藏

| 项目 | 内容 |
|------|------|
| 领导人 | 达赖喇嘛十三世 |
| 控制区 | 西藏、西康一部 |
| 军队 | 藏军（约 1 万） |
| 特点 | 政教合一、实际独立 |

- 西藏自清末以来处于实质独立状态。
- 英国在西藏有传统影响力，供应武器。
- 与中央政府在康区（西康）有领土争端，1931 年有军事摩擦。
- 对日本侵华态度冷淡。

## 五、对 AI 决策的参考意义

### 各势力战略优先级

| 势力 | 短期核心目标 | 长期核心目标 | 最大威胁 |
|------|-------------|-------------|---------|
| KMT | 剿共、巩固中央权威 | 统一全国 | 日军入侵、中共壮大 |

| JPN | 巩固东北、分裂华北 | 全面侵华 | 苏联干涉 |
| NEA | 保存实力、收复东北 | 打回老家 | 日军、蒋介石吞并 |
| SHX | 保境安民 | 维护山西独立 | 日军、中央军 |
| GXC | 壮大桂系 | 问鼎中央 | 中央军吞并 |
| SCC | 统一四川 | 走出四川 | 内部混战、中央军 |
| MA | 维持地盘 | 维护回民利益 | 中央军、红军 |
| XJ | 镇压叛乱 | 维持统治 | 苏联、民族起义 |
| TIB | 维持独立 | 维持独立 | 中央政府 |

### 城市战略价值公式

```
attackScore = cityLevel * 3 + industry * 2 + food - fort
```

数值越高，AI 越应优先进攻/争夺该城市。

### AI 行为原则

1. **保存实力优先**：除 JPN 外，所有势力都应优先保存核心军队。
2. **利用外部矛盾**：KMT 和 CCP 相互牵制时，第三方可渔翁得利。
3. **地形决定战术**：山地防守加成 50%，攻城防御加成 40%，AI 应避免在不利地形强攻。
4. **工业决定战争潜力**：高工业城市（上海、沈阳、武汉、天津）是决定长期战争胜负的关键。
5. **粮食决定动员能力**：饥荒地区无法支撑大规模征兵。
